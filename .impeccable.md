## Design Context

### Users
PokeAPI 面向需要稳定接入多家 AI 模型的开发者、AI 应用团队、API 中转站管理员与企业采购/运维人员。典型场景是快速判断服务是否可信，注册后获取 API Key，并把现有 OpenAI / Anthropic / CLI 工具的 Base URL 指向平台。

### Brand Personality
克制、锋利、可信。界面应该像独立设计工作室的黑白作品集：少量文字、强烈排版、细线结构和明确动效，让技术平台显得稳定、有审美、可长期依赖。

### Aesthetic Direction
公开入口采用近黑与象牙白主色、超大错位展示标题、mono 小标签、细线分隔和黄绿色信号色。主页以三幕正常滚动驱动的 sticky 编辑舞台讲清真实模型、协议接入与透明账单；登录/注册使用路由感知的协议轨道，文档主页使用同源编辑式结构。所有轨道、节点与信号轨迹均由原创 CSS / SVG 程序化生成，不复制任何参考站素材或文案；触摸设备、省流量和 `prefers-reduced-motion` 环境使用静态降级。

### Design Principles
1. 信息先于装饰：每个视觉层都必须服务“稳定接入、可观测、低成本”的判断。
2. 黑白编辑感：避免常见 SaaS 渐变卡片和霓虹科技风，用纸张、墨色、细线和空间建立高级感。
3. 动效克制但明确：页面进入、悬停、复制反馈和弹层使用快速、自然的 transform/opacity 动效。
4. 公开入口统一：主页、登录、注册共享同一套排版、纹理、按钮、表单和底部导航语言。
5. 原创资产：不引用参考站图片、视频或品牌素材；公开入口优先使用文字、真实端点和产品数据，不用无意义装饰图。
6. 诚实隐私：仅承诺处理账户、路由、计费与安全所必需的数据；明确不保存提示词/回复正文、不出售、不用于广告画像或模型训练，避免“完全零采集”等与服务运行冲突的绝对表述。
